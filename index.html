Skills and Expertise

I've been developing skills to quickly code user friendly, intelligent, elegantly designed websites

Frontend Development

I write code that conforms to the W3C's web standards and semantic best practices. I use technologies like HTML5 / Jade, SASS, Javascript and jQuery, to transform static artwork into mobile first, pixel prefect, interactive and fully functional interfaces; exactly the way your designers intend them to be.

Responsive Web Development

How will we ensure your website looks great on multiple devices; including those that haven't even been invented yet? I build using a responsive grid system mobile first. I have experience with frameworks such as Bootstrap and Bourbon Neat and am practiced with flex box. All of these technologies ensure sure that layouts are now quicker, smarter and more future-proof.

Design and Fine Arts Integrated with User Experience

My education in fine arts and design help me compose beautiful creative sites that direct and aide the users' experience. I know Sketch and inVision and can create wireframes for client and developer projects. I also enjoy creating icons, logos and other assets with Sketch and Adobe.

Selected Work

Arcana Tarot

This website provides users with a tarot card reading anywhere and anytime. I used the project to polish my javascript, jQuery, Sass, and Sketch skills. All of the content is design and produced my me; inculding the logo, icons, card backs and card fronts. I drew all of the assets in Sketch.
This is any ongoing project at the moment. I'm still working on the card front images and am polishing the sophistication of the javascript logic. My goal is to write javascript that will provide readings based on parsing the users input. I am also refining the animations.

Frog Simone and Trytrist

Two games for children ages 6-12 based off the memory game, simon, and the strategy game, candy crush. The focus was to take well known games and create user friendly and pleasing animations.
Technologies used: Ember.js, JavaScript, Node, Bower, Gulp, HTML5, Sass, Sketch

BarterBay

This is a website used to barter for goods and services. Users can create and update profiles, list goods and services and tag things they are interested in trading for. Other users can rate their experiences with users and that rating will show up on profiles.
Technologies used: Ruby, Sinatra, SQL, HTML, ActiveRecord, Sass
